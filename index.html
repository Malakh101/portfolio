Personal Project · Public Dataset

Medical Insurance Cost Analysis - Power BI Dashboard

Records1,338
Avg Charge$13.27K
Top 5% Share100% smokers
Challenge

Build a from-scratch BI dashboard on a real (public, CC0-licensed) health-insurance dataset, to demonstrate the same skill set used on employer dashboards without touching any confidential data.

Approach
  • Modeled the data in Power BI, including a DAX-calculated field flagging the top 5% highest-cost policyholders by percentile threshold.
  • Built KPI cards, a regional cost comparison, a smoker/gender cost-gap breakdown, and a cost-concentration view.
  • Corrected an early version that summed by region without accounting for uneven group sizes - switched to average-per-region so the comparison reflects real cost differences, not headcount.
Outcome

Despite smokers being only ~20% of the dataset, they make up 100% of the top 5% highest-cost policyholders - averaging $46.59K versus $11.51K for everyone else. Southeast carries the highest regional average ($14.7K), 8–19% above other regions.
